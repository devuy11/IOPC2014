Z= floor ((a!/b)) . Is Z odd or even?

Input Format ::

First line contains T. Next T line follows each line containg a and b.

Output Format::

If Z is even output "Even" without quotes otherwise output "Odd"

Sample Input:

1

3 6

Sample Output ::

Odd
Constraints:

a<=1e5

b<=1e18 and b>=1

testcases<=100
